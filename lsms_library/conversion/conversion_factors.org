* Purpose

Phase 2 of currency handling (see =slurm_logs/DESIGN_currency_conversion_2026-06-17.org=).
This file is the single source of truth for converting nominal local-currency
monetary values (=food_expenditures.Expenditure=, =assets.Value=, ...) to
comparable bases.  =lsms_library.conversion.convert()= reads the
=conversion_factors= table below via =df_from_orgfile=; the prose and the
generator that produced it live alongside the numbers, so each column is
documented next to the data it describes.

* Reading the table

- Key :: =(Country, Date)=, where =Country= is a repo country name (so =GHA=
  appears as =GhanaLSS=) and =Date= is a calendar year.  =Currency= is carried
  as a convenience column.  The key is *Country*, not *Currency*, because PPP
  and CPI are country-specific even inside a currency union: Mali and Niger
  both use XOF but have different price levels, hence different PPP factors
  (compare their =PPP-2017= rows) -- one of which would be silently wrong if a
  single XOF row served both.
- Orientation :: every factor column is *LCU per target unit*, so

  \[ x_{\text{target}} = x_{\text{LCU}} \,/\, \text{factor} . \]

  A new conversion is a new column following this rule.
- Missing :: an empty/=---= cell is =pd.NA=; =convert()= propagates it (no
  silent fallback).

* Column glossary

- Currency :: ISO 4217 alpha-3 (matches =data_info.yml= =Currency:=).
- FX :: LCU per current USD.  WDI =PA.NUS.FCRF= (official, period average).
  Nominal USD at market rates.
- PPP-2011 / PPP-2017 / PPP-2021 :: LCU per *constant-base-year PPP
  international dollar* (real PPP).  Consumption PPP =PA.NUS.PRVT.PP=
  benchmarked to the named year, with intertemporal deflation by the local CPI
  baked in: for base \(b\),

  \[ \text{PPP-}b(c,t) = \mathrm{PPP}_c(b)\,\cdot\,\frac{\mathrm{CPI}_c(t)}{\mathrm{CPI}_c(b)} . \]

  =PPP-2017= is the headline welfare measure (World Bank poverty-line basis).
- USD-real-2017 :: LCU per constant-2017 USD; =FX= composed with the US CPI
  (=FP.CPI.TOTL= for USA), \(\text{FX}(c,t)\cdot \mathrm{CPI}_{US}(t)/\mathrm{CPI}_{US}(2017)\).
  Consumer-basket deflator (standard for consumption/wages; a GDP-deflator
  variant would be a separate column).  For *cross-country* welfare prefer
  =PPP-2017=: market FX understates poorer countries' real consumption
  (Balassa-Samuelson).
- LCU-real-2017 :: LCU per constant-2017 local currency, \(\mathrm{CPI}_c(t)/\mathrm{CPI}_c(2017)\)
  (local CPI only; intra-country temporal comparability).
- CPI :: local consumer price index =FP.CPI.TOTL= (helper; not a conversion
  target).

* Redenomination caveat (v1 scope)

The values below come from WDI, which publishes each series in the *current*
currency unit.  For the four pre-redenomination waves whose ISO code differs
from the modern default -- GhanaLSS <= 2005-06 (GHC), Tajikistan 1999 (TJR),
Azerbaijan 1995 (AZM), Serbia-and-Montenegro 2002 (YUM) -- a row keyed on the
calendar year would carry the *rebased* (modern-currency) factor, which is
10^3-10^4 off the contemporaneous data.  =convert()= therefore returns =pd.NA=
+ a warning for any wave whose phase-1 currency code is a redenomination
override (data-driven from =data_info.yml= =Currency:= =overrides=), regardless
of what this table holds.  Sourcing contemporaneous-currency factors (IMF IFS)
for those rows is deferred.

* Provenance / regeneration

Source: World Bank WDI, accessed 2026-06-17.  The table is a versioned
snapshot, not a live query; =convert()= only reads it.  To refresh the vintage
or extend coverage (more countries/years/columns), run the block below in
Emacs (=C-c C-c=) -- it pulls WDI over plain HTTP (no extra dependency),
composes the columns, and rewrites the table.  Extend =ISO3= for the full
country set; add a column by adding it to =COLS= and =factors()=.

#+begin_src python :results output :exports both
import json, urllib.request

# repo country name -> ISO3 (WDI).  Extend for the full country set.
ISO3 = {"Uganda":"UGA","Mali":"MLI","Niger":"NER","GhanaLSS":"GHA",
        "Tanzania":"TZA","Nigeria":"NGA","Ethiopia":"ETH","Malawi":"MWI"}
CUR  = {"Uganda":"UGX","Mali":"XOF","Niger":"XOF","GhanaLSS":"GHS",
        "Tanzania":"TZS","Nigeria":"NGN","Ethiopia":"ETB","Malawi":"MWK"}
YEARS, BASES = range(2005, 2023), (2011, 2017, 2021)
FX, PPP, CPI = "PA.NUS.FCRF", "PA.NUS.PRVT.PP", "FP.CPI.TOTL"
COLS = ["FX","PPP-2011","PPP-2017","PPP-2021","USD-real-2017","LCU-real-2017","CPI"]

def wdi(indicator, isos):
    url = (f"https://api.worldbank.org/v2/country/{';'.join(isos)}/indicator/"
           f"{indicator}?date={YEARS.start}:{YEARS.stop-1}&format=json&per_page=20000")
    with urllib.request.urlopen(url, timeout=60) as r:
        payload = json.load(r)
    return {(row["countryiso3code"], int(row["date"])): float(row["value"])
            for row in (payload[1] or []) if row["value"] is not None}

isos = sorted(set(ISO3.values()) | {"USA"})
fx, ppp, cpi = wdi(FX, isos), wdi(PPP, isos), wdi(CPI, isos)
g = lambda d, iso, yr: d.get((iso, yr))

def factors(iso, t):
    row = {"FX": g(fx, iso, t)}
    c_t = g(cpi, iso, t)
    for b in BASES:
        pb, cb = g(ppp, iso, b), g(cpi, iso, b)
        row[f"PPP-{b}"] = pb * c_t / cb if (pb and c_t and cb) else None
    us_t, us_b = g(cpi, "USA", t), g(cpi, "USA", 2017)
    row["USD-real-2017"] = row["FX"] * us_t / us_b if (row["FX"] and us_t and us_b) else None
    c_b = g(cpi, iso, 2017)
    row["LCU-real-2017"] = c_t / c_b if (c_t and c_b) else None
    row["CPI"] = c_t
    return row

def fmt(x):
    if x is None: return "---"
    if abs(x) >= 100: return f"{x:.1f}"
    if abs(x) >= 1:   return f"{x:.3f}"
    return f"{x:.5f}"

hdr = ["Country","Currency","Date"] + COLS
print("| " + " | ".join(hdr) + " |")
print("|" + "+".join(["-"*(len(h)+2) for h in hdr]) + "|")
for name in sorted(ISO3):
    for t in YEARS:
        row = factors(ISO3[name], t)
        if all(row[c] is None for c in COLS): continue
        print("| " + " | ".join([name, CUR[name], str(t)] + [fmt(row[c]) for c in COLS]) + " |")
#+end_src
