* Purpose

Phase 2 of currency handling (see =slurm_logs/DESIGN_currency_conversion_2026-06-17.org=).
This file is the single source of truth for converting nominal local-currency
monetary values (=food_expenditures.Expenditure=, =assets.Value=, ...) to
comparable bases.  =lsms_library.conversion.convert()= reads the
=conversion_factors= table below via =df_from_orgfile=; the prose and the
generator that produced it live alongside the numbers, so each column is
documented next to the data it describes.

* Reading the table

- Key :: =(Country, Date)=, where =Country= is a repo country name (so =GHA=
  appears as =GhanaLSS=).  The key is *Country*, not *Currency*, because PPP and
  CPI are country-specific even inside a currency union: Mali and Niger both use
  XOF but have different price levels, hence different =PPP-2017= rows -- one of
  which would be silently wrong if a single XOF row served both.  =Currency= is
  a convenience column.
- Date :: an *ISO-8601 partial date* at any resolution -- =2019=, =2019-08=, or
  =2019-08-15= -- resolved to the start of the period (=2019= -> =2019-01-01=,
  =2019-08= -> =2019-08-01=).  Rows are currently yearly; finer rows can be
  added where a wave straddles volatile months and the lookup will prefer them.
- Time match (dynamic / as-of) :: =convert()= matches each row to the *most
  recent factor at or before its interview date* (=merge_asof= backward), so a
  household interviewed in 2019 within a =2018-19= wave gets the 2019 factor and
  one interviewed in 2018 gets 2018 -- per household, exact.  Households with no
  =interview_date= (and the 6 countries lacking the table) fall back to the
  wave's nominal date (wave year -> Jan 1).
- Orientation :: every factor column is *LCU per target unit*, so

  \[ x_{\text{target}} = x_{\text{LCU}} \,/\, \text{factor} . \]

  A new conversion is a new column following this rule.
- Missing :: an empty/=---= cell, or a query date before a country's earliest
  row, yields =pd.NA= (no silent fallback).

* Column glossary

- Currency :: ISO 4217 alpha-3 (matches =data_info.yml= =Currency:=).
- FX :: LCU per current USD.  WDI =PA.NUS.FCRF= (official, period average).
  Nominal USD at market rates.
- PPP-2011 / PPP-2017 / PPP-2021 :: LCU per *constant-base-year PPP
  international dollar* (real PPP).  Consumption PPP =PA.NUS.PRVT.PP=
  benchmarked to the named year, with intertemporal deflation by the local CPI
  baked in: for base \(b\),

  \[ \text{PPP-}b(c,t) = \mathrm{PPP}_c(b)\,\cdot\,\frac{\mathrm{CPI}_c(t)}{\mathrm{CPI}_c(b)} . \]

  =PPP-2017= is the headline welfare measure (World Bank poverty-line basis).
- USD-real-2017 :: LCU per constant-2017 USD; =FX= composed with the US CPI
  (=FP.CPI.TOTL= for USA), \(\text{FX}(c,t)\cdot \mathrm{CPI}_{US}(t)/\mathrm{CPI}_{US}(2017)\).
  Consumer-basket deflator (standard for consumption/wages; a GDP-deflator
  variant would be a separate column).  For *cross-country* welfare prefer
  =PPP-2017=: market FX understates poorer countries' real consumption
  (Balassa-Samuelson).
- LCU-real-2017 :: LCU per constant-2017 local currency, \(\mathrm{CPI}_c(t)/\mathrm{CPI}_c(2017)\)
  (local CPI only; intra-country temporal comparability).
- CPI :: local consumer price index =FP.CPI.TOTL= (helper; not a conversion
  target).

* Redenomination handling

WDI publishes each series in the *current* currency unit, so a naive row keyed
on the calendar year would carry the *rebased* factor -- 10^3-10^4 off the
contemporaneous nominal data for a pre-reform wave (e.g. GhanaLSS 2005-06 is in
old cedi GHC, ~10^4 the new cedi GHS).  Across a redenomination the
contemporaneous old-currency factor is *exactly* the rebased factor times the
redenomination scale (CPI is an index and is left raw), so the generator below
rescales the pre-reform rows accordingly and tags them with the old ISO code:

- GhanaLSS :: years < 2007 in GHC (x10,000).  Sanity: 1987 FX 153.7, 2005 FX
  9052 GHC/USD -- matches the historical official rate; 2007+ are GHS.
- Tajikistan :: 1999 in TJR (x1,000).  FX 1238 TJR/USD; PPP-/LCU-real columns
  are NaN because WDI has no 1999 Tajikistan CPI (no silent guess).
- Azerbaijan :: years < 2006 in AZM (x5,000).  1995 FX 4413 AZM/USD; 2006+ AZN.
- Serbia and Montenegro :: YUM/CSD/RSD are 1:1, so the 2002 row needs no rescale.

So these waves now *convert* (the engine just reads the contemporaneous factor;
a genuinely missing/NaN factor still yields =pd.NA= + a warning).  The IMF IFS
API was unreachable from the build host, but WDI itself reaches back to 1987 for
these series, and rebased x scale is the exact contemporaneous value -- so IFS
was not needed.  The FX/PPP/USD-real/LCU-real columns therefore show a deliberate
10^3-10^4 discontinuity at each reform year; that is correct (the unit changed).

* Provenance / regeneration

Source: World Bank WDI, accessed 2026-06-17.  The table is a versioned
snapshot, not a live query; =convert()= only reads it.  To refresh the vintage
or extend coverage (more countries/years/columns), run the block below in
Emacs (=C-c C-c=) -- it pulls WDI over plain HTTP (no extra dependency),
composes the columns, and rewrites the table.  =ISO3=/=CUR= cover all 34 repo
countries (2000-2023); widen =YEARS= for a longer span, or add a new conversion
by adding a column to =COLS= and =factors()=.

"""Generate conversion_factors org-table rows from World Bank WDI (no extra deps)."""
import json, urllib.request

# repo country name -> ISO3 (WDI).  Two repo entries share an ISO3
# (EthiopiaRHS->ETH, "Serbia and Montenegro"->SRB): each still gets its own rows.
ISO3 = {
    "Albania": "ALB", "Armenia": "ARM", "Azerbaijan": "AZE", "Benin": "BEN",
    "Burkina_Faso": "BFA", "Cambodia": "KHM", "China": "CHN", "CotedIvoire": "CIV",
    "Ethiopia": "ETH", "EthiopiaRHS": "ETH", "GhanaLSS": "GHA", "Guatemala": "GTM",
    "Guinea-Bissau": "GNB", "Guyana": "GUY", "India": "IND", "Iraq": "IRQ",
    "Kazakhstan": "KAZ", "Kosovo": "XKX", "Liberia": "LBR", "Malawi": "MWI",
    "Mali": "MLI", "Nepal": "NPL", "Niger": "NER", "Nigeria": "NGA",
    "Pakistan": "PAK", "Senegal": "SEN", "Serbia": "SRB",
    "Serbia and Montenegro": "SRB", "South Africa": "ZAF", "Tajikistan": "TJK",
    "Tanzania": "TZA", "Timor-Leste": "TLS", "Togo": "TGO", "Uganda": "UGA",
}
# modern default ISO 4217 (cosmetic Currency column; matches data_info.yml).
CUR = {
    "Albania": "ALL", "Armenia": "AMD", "Azerbaijan": "AZN", "Benin": "XOF",
    "Burkina_Faso": "XOF", "Cambodia": "KHR", "China": "CNY", "CotedIvoire": "XOF",
    "Ethiopia": "ETB", "EthiopiaRHS": "ETB", "GhanaLSS": "GHS", "Guatemala": "GTQ",
    "Guinea-Bissau": "XOF", "Guyana": "GYD", "India": "INR", "Iraq": "IQD",
    "Kazakhstan": "KZT", "Kosovo": "EUR", "Liberia": "LRD", "Malawi": "MWK",
    "Mali": "XOF", "Nepal": "NPR", "Niger": "XOF", "Nigeria": "NGN",
    "Pakistan": "PKR", "Senegal": "XOF", "Serbia": "RSD",
    "Serbia and Montenegro": "CSD", "South Africa": "ZAR", "Tajikistan": "TJS",
    "Tanzania": "TZS", "Timor-Leste": "USD", "Togo": "XOF", "Uganda": "UGX",
}
# Redenominations within the sample: rows before the reform year are rescaled to
# the CONTEMPORANEOUS (pre-reform) currency = WDI-rebased value x scale, so the
# factor matches that wave's nominal data (in the old unit).  CPI is an index
# (scale-free) and is left raw.  YUM/CSD/RSD are 1:1, so Serbia needs no entry.
REDENOM = {"GhanaLSS": (2007, 10000), "Tajikistan": (2000, 1000), "Azerbaijan": (2006, 5000)}
# Contemporaneous ISO 4217 for pre-reform rows (so the Currency column is honest).
OLD_CODE = {"GhanaLSS": "GHC", "Tajikistan": "TJR", "Azerbaijan": "AZM"}
# Extra pre-2000 survey-wave years to emit beyond the default range.
PRE_REFORM_EXTRA = {"GhanaLSS": [1987, 1988, 1991, 1998], "Tajikistan": [1999], "Azerbaijan": [1995]}

YEARS, BASES = range(2000, 2024), (2011, 2017, 2021)   # default emission range
PULL = range(1987, 2024)                               # fetch wide for old years
FX, PPP, CPI = "PA.NUS.FCRF", "PA.NUS.PRVT.PP", "FP.CPI.TOTL"
COLS = ["FX", "PPP-2011", "PPP-2017", "PPP-2021", "USD-real-2017", "LCU-real-2017", "CPI"]


def wdi(indicator, isos):
    url = (f"https://api.worldbank.org/v2/country/{';'.join(isos)}/indicator/"
           f"{indicator}?date={PULL.start}:{PULL.stop-1}&format=json&per_page=20000")
    with urllib.request.urlopen(url, timeout=120) as r:
        payload = json.load(r)
    return {(row["countryiso3code"], int(row["date"])): float(row["value"])
            for row in (payload[1] or []) if row["value"] is not None}


isos = sorted(set(ISO3.values()) | {"USA"})
fx, ppp, cpi = wdi(FX, isos), wdi(PPP, isos), wdi(CPI, isos)
g = lambda d, iso, yr: d.get((iso, yr))


def factors(iso, t):
    row = {"FX": g(fx, iso, t)}
    c_t = g(cpi, iso, t)
    for b in BASES:
        pb, cb = g(ppp, iso, b), g(cpi, iso, b)
        row[f"PPP-{b}"] = pb * c_t / cb if (pb and c_t and cb) else None
    us_t, us_b = g(cpi, "USA", t), g(cpi, "USA", 2017)
    row["USD-real-2017"] = row["FX"] * us_t / us_b if (row["FX"] and us_t and us_b) else None
    c_b = g(cpi, iso, 2017)
    row["LCU-real-2017"] = c_t / c_b if (c_t and c_b) else None
    row["CPI"] = c_t
    return row


def fmt(x):
    if x is None: return "---"
    if abs(x) >= 100: return f"{x:.1f}"
    if abs(x) >= 1:   return f"{x:.3f}"
    return f"{x:.5f}"


hdr = ["Country", "Currency", "Date"] + COLS
print("| " + " | ".join(hdr) + " |")
print("|" + "+".join(["-"*(len(h)+2) for h in hdr]) + "|")
for name in sorted(ISO3):
    iso = ISO3[name]
    years = sorted(set(YEARS) | set(PRE_REFORM_EXTRA.get(name, [])))
    for t in years:
        row = factors(iso, t)
        code = CUR[name]
        if name in REDENOM:                       # contemporaneous old-currency rescale
            ry, scale = REDENOM[name]
            if t < ry:
                code = OLD_CODE[name]
                for c in COLS:
                    if c != "CPI" and row[c] is not None:
                        row[c] *= scale
        if all(row[c] is None for c in COLS): continue
        print("| " + " | ".join([name, code, str(t)] + [fmt(row[c]) for c in COLS]) + " |")
