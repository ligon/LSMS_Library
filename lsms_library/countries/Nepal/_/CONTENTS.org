Nepal has three NLSS (Nepal Living Standards Survey) waves:
- 1995-96 (NLSS I) -- WB catalog 2301
- 2003-04 (NLSS II) -- WB catalog 1000
- 2010-11 (NLSS III) -- WB catalog (check SOURCE.org)

** No .dta files in repository
As of 2026-04-02, no source data files have been DVC-tracked or
downloaded.  The =data_info.yml= and =data_scheme.yml= configs exist
(household_roster, cluster_features, individual_education,
food_acquired, interview_date, assets) but all features return empty
DataFrames because the source .dta files are missing.

** Data sources
The World Bank does NOT host Nepal microdata directly.  Both WB
catalog pages redirect to the Nepal NSO.  Data must be obtained from
the NSO NADA portal.

- *NSO NADA Catalog*: https://microdata.nsonepal.gov.np/index.php/catalog
  - NLSS I (1995-96): catalog id 7, 77 .dta files, 942 variables
  - NLSS II (2003-04): catalog id 9, 79 .dta files, 896 variables
  - NLSS III (2010-11): catalog id 37, 48 .dta files, 999 variables
- Access type: "Licensed" --- free registration required at the NSO portal
- File formats: Stata (.dta)
- File naming: =z*= (NLSS I), =Z*= (NLSS II), =xh*_s*= (NLSS III)
- WB catalog 2301 and 1000 redirect to =nsonepal.gov.np= (no direct download)
