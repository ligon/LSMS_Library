- 1995-96 (NLSS I) -- WB catalog 2301
- 2003-04 (NLSS II) -- WB catalog 1000
- 2010-11 (NLSS III) -- WB catalog (check SOURCE.org)

** Data sources
The World Bank does NOT host Nepal microdata directly.  Both WB
catalog pages redirect to the Nepal NSO.  Data must be obtained from
the NSO NADA portal.

- *NSO NADA Catalog*: https://microdata.nsonepal.gov.np/index.php/catalog
  - NLSS I (1995-96): catalog id 7, 77 .dta files, 942 variables
  - NLSS II (2003-04): catalog id 9, 79 .dta files, 896 variables
  - NLSS III (2010-11): catalog id 37, 48 .dta files, 999 variables
- Access type: "Licensed" --- free registration required at the NSO portal
- File formats: Stata (.dta)
- File naming: =z*= (NLSS I), =Z*= (NLSS II), =xh*_s*= (NLSS III)
- WB catalog 2301 and 1000 redirect to =nsonepal.gov.np= (no direct download)

* Sampling Design

All three NLSS waves use a two-stage stratified cluster sample.
PSUs are wards (=WWW= in waves I--II, =xhpsu= in wave III).

** Strata and PSU identifiers

| Wave    | PSU var | Strata var | Urban/Rural var | HH weight var |
|---------+---------+------------+-----------------+---------------|
| 1995-96 | WWW     | stratum    | urbrural        | weight        |
| 2003-04 | WWW     | stratum    | urbrural        | weight        |
| 2010-11 | xhpsu   | stratum    | urbrur          | wt_hh         |

** Weight types

NLSS I and II provide a single household weight (raising factor).
NLSS III provides both =wt_hh= (household weight) and =wt_ind=
(individual weight = hh weight * hh size).  Since =sample= is
household-level, both =weight= and =panel_weight= map to =wt_hh=.
There is no rotating panel or refreshment sample in the NLSS series.

** Source files

- NLSS I (1995-96): =c1_nlss_public.dta= (HH-level, has WWWHH + weight)
  joined with =sample_hh.dta= (PSU-level, has stratum + urbrural)
- NLSS II (2003-04): =c2_nlss_public.dta= (HH-level) joined with
  =sample.dta= (PSU-level)
- NLSS III (2010-11): =sample.dta= (HH-level, 7180 cases, has all
  sampling variables)
