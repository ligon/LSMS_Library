#+title: Categorical Mapping for Nigeria

* Food Items

  Food item codes changed between wave 1 (2010-11) and wave 2 (2012-13),
  when the survey expanded from ~80 to ~130 items.  The Preferred Label
  is the most specific available label; the Aggregate Label groups
  sub-varieties (e.g. "Maize (shelled)" and "Maize (unshelled)" both
  aggregate to "Maize").

* Units

  Unit codes changed significantly across waves.  Wave 1 (2010-11) used
  6 simple codes (Kg, g, l, ml, piece, other).  Wave 2 (2012-13) added
  local unit names (mudu, derica, congo, etc.) with codes 1-16.
  Waves 3-4 (2015-16, 2018-19) restructured to codes with size
  modifiers (small/medium/large) using codes 1-122 and 900.

  Note: the same code number can mean different things across waves
  (e.g., code 10 = "medium derica" in wave 2 but "bin/basket" in
  waves 3-4).  The per-wave columns record the wave-specific meaning.
